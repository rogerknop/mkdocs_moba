* Standard Raspberry Pi Image aufspielen
* Feste IP vergeben
* Rocrail installieren per UNZIP aktuelles Package
* sudo systemctl enable --now cron
* crontab -e: @reboot /home/pi/Rocrail/startrocrail.sh
* nas mounten
